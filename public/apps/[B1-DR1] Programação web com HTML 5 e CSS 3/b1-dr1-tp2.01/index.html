Tipos de Seletores CSS

Seletor por TAG

Seleciona os elementos de acordo com o nome da tag

div { }

Ex: Todas as tags "pre" foram estilizadas com fundo

Seletor por ID

Seleciona nós baseados no atributo "id", devendo haver apenas um único elemento para cada id 

#id { }

Ex: O seguinte elemento: Exemplo foi estilizado por ID

Seletor por CLASSE

Seleciona os elementos a partir do atributo "class".

.nome-da-classe { }

Ex: Cada cartão foi estilizado por classe

Seletor UNIVERSAL

Seleciona todos os elementos ou nós

* { }

Ex: Todos os elementos tiveram sua fonte alterada para "sans-serif"

Seletor por ATRIBUTO

Seleciona todos os elementos que possuam um determinado atributo ou que o valor do atributo seja específico 

[atributo] { }   tag[atributo=valor] { }

Ex: Um link que abre na mesma guia não terá seu estilo aplicado enquanto um link que abre uma nova guia terá.

Seletor por PSEUDO-CLASSE

Combinado com outro seletor, abrange elementos a partir do seu estado interno, como :hover quando está com o mouse sobre ele ou :focus quando está em destaque 

tag:hover { } #id:focus { }

Ex: Um terá seu estilo alterado apenas quando estiver com o mouse sobre ele
