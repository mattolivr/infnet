1. Como funciona a Web e o que é uma página web

A Web é uma rede mundial de computadores que permite a troca de informações de forma global por meio de páginas. Um computador qualquer pode digitar um endereço URL e acessar um servidor por meio do protocolo HTTP, assim recebendo suas informações a partir de uma página exibida pelo navegador.

As páginas web são arquivos no formato HTML armazenados nos servidores que, com o auxílio de CSS e JavaScript, exibem para o usuário o conteúdo que ele acessou

2. Quais os papéis das linguagens HTML e CSS numa página web

O HTML é a estrutura base de uma página web, sendo comparado ao "esqueleto" dela. Ele dita a estrutura da página, seu layout, textos, links, imagens etc. por meio de "tags".

Já o CSS exerce o papel de estilização da página, aplicando estilo às tags HTML, alterando sua forma, cor e comportamento.

3. O que são ambientes de programação

Um ambiente de programação é um software que proporciona ferramentas de desenvolvimento de código para a edição do código-fonte, testes e depuração do sistema desenvolvido.
