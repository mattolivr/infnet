# Caderno

O **Caderno** é um portfólio desenvolvido para facilitar a execução e entrega de atividades acadêmicas do autor para o curso de Engenharia de Software do Instituto Infnet. O projeto surgiu como uma iniciativa para expor as atividades realizadas no curso, simultaneamente automatizando processos relacionados ao desenvolvimento e entrega.

## Funcionalidades Principais

- **Visualização de Disciplinas e Exercícios**: interface responsiva que permite a visualização dos enunciados e exercícios executados.
- **Extração Automática de Dados**: scripts que permitem a extração do conteúdo dos enunciados diretamente da página do portal da faculdade, a partir dos arquivos HTML.
